# Highlight Analysis Skill

## 분석 프레임워크

### 1단계: 전체 트랜스크립트 스캔

SRT 파일을 읽고 다음을 파악:
- 전체 주제 / 핵심 메시지
- 화자 수 및 역할
- 전체 길이 대비 인사이트 밀도 구간

### 2단계: 후킹 포인트 스코어링

각 30초 구간에 대해 5가지 기준으로 1~5점 채점:

| 기준 | 1점 | 5점 |
|------|-----|-----|
| 감정 전환 | 평이한 설명 | 놀라움/반전/강한 주장 |
| 공유성 | 맥락 없이 이해 불가 | 한 문장으로 캡처 가능 |
| 궁금증 | 결론이 명확 | "왜? 어떻게?" 유발 |
| 구체성 | 추상적 개념 | 수치/사례/이름 포함 |
| 시각성 | 말로만 설명 | 영상으로 보여줄 게 있음 |

총점 20/25 이상 → 후킹 후보

### 3단계: 숏폼 구성

후킹 후보별 구성안 설계:

```
[후킹] 3~5초 — 가장 임팩트 있는 1~2문장
  ↓
[맥락] 30~40초 — 후킹을 이해하기 위한 배경
  ↓
[클라이맥스] 10~15초 — 후킹 포인트 전후 맥락 포함 원본 재생
  ↓
[마무리] 5초 — 여운 또는 다음 예고
```

### 4단계: 출력 형식

각 후보마다:

```
## 후보 N: [후킹 문장 요약]

**후킹 점수:** 22/25
**타임코드:** 12:34~13:28 (원본 기준)
**후킹 대사:** "실제로 AI가 대체하는 건 직업이 아니라 태도입니다"
**후킹 이유:** 기존 "AI가 일자리를 뺏는다" 통념을 뒤집는 반전

| 구간 | 원본 타임코드 | 길이 | 내용 |
|------|-------------|------|------|
| 후킹 | 12:45~12:50 | 5초 | 핵심 반전 문장 |
| 맥락 | 12:00~12:45 | 45초 | AI와 직업의 관계 설명 |
| 클라이맥스 | 12:50~13:05 | 15초 | 구체 사례 |
| 마무리 | - | 5초 | 엔딩 |

**예상 길이:** 70초
**공유 문장:** "AI가 대체하는 건 직업이 아니라 태도"
```

## 주의사항

- 후킹은 반드시 원본 대사에서 추출 — 창작 금지
- 타임코드는 SRT 기준 정확히 매칭
- 맥락 구간은 후킹을 이해하기에 충분해야 함 (너무 짧으면 의미 전달 실패)
- 한 영상에서 겹치는 구간이 있는 후보는 별도 표기

## 후킹 편집 기법 (리서치 기반)

### 첫 3초 체크리스트

- 프레임 1에 움직임이 있어야 함 (정적 화면 금지)
- 1초 이내에 자막 텍스트 노출
- 0.3초 이상 무음 구간 제거
- "인사"나 "안녕하세요"로 시작하지 않음 — 바로 본론

### 후킹 유형별 효과 (2025 데이터)

| 순위 | 유형 | 리텐션 향상 | 패턴 |
|------|------|------------|------|
| 1 | 문제/해결 | +48% | "~에 대해 가장 많이 하는 실수" |
| 2 | 결과 먼저 | +32% | 최종 결과를 먼저 보여주고 과정 설명 |
| 3 | 대담한 주장 | +35% | 동의/반대 모두 시청 유지 |
| 4 | 호기심 갭 | +29% | "이 방법 아는 사람 거의 없습니다" |
| 5 | 패턴 인터럽트 | +27% | 예상 못한 시각/청각 전환 |

### 한국 시장 후킹 패턴

- "이걸 모르면 손해입니다" — 손실 회피
- "99%가 모르는 [X] 비법" — 독점/호기심
- "실제로 해봤습니다" — 진정성
- "왜 아무도 이걸 안 알려줬지?" — 정보 격차
- "이거 하나면 끝입니다" — 단순화

### 숏폼 구조 최적화

```
[0-2초]  HOOK    — 패턴 인터럽트 + 핵심 문장
[2-8초]  TEASE   — 가치 암시, 답은 보류
[8-45초] BODY    — 2-3초마다 컷 전환, 중간에 리훅
[45-55초] PAYOFF — 약속한 답/인사이트 전달
[55-60초] LOOP   — 엔딩이 시작으로 자연스럽게 이어지도록
```

### 리텐션 핵심 지표

- **75%+ 시청 완료율** → 알고리즘 3배 노출 증가
- **50% 지점**에서 반드시 새 정보 또는 리훅 삽입
- 4초 이상 같은 화면 유지 금지 — 컷 전환 또는 줌
